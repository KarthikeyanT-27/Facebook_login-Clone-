# Facebook_login-Clone-

A pixel-accurate clone of the Facebook login page built with pure HTML, CSS, and JavaScript.

## 📁 Files
```
facebook-login/
├── index.html   # Page structure
├── style.css    # Styling & layout
├── app.js       # Form validation & interactivity
└── README.md
```

## 🚀 Run
Just open `index.html` in any browser — no setup needed.

## ✨ Features
- Facebook logo, collage illustration & hero text
- Login form with validation & loading state
- "Create new account" modal with full sign-up form
- Responsive design (mobile-friendly)
- No libraries or frameworks
